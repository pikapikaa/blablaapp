задание REact native
Использовать следующее API - https://dummyjson.com/docs/products
Нижняя навигация - главная страница, профиль.

1. В приложении должны быть авторизация, выход из профиля и просмотр профиля. Все из dummyjson.
2. Сначала появляется окно авторизации. Выбираете любого пользователя из ответа https://dummyjson.com/users и авторизуетесь. Без авторизации функционал приложения не должен быть доступен.
3. В шапке должен быть значок корзины. При добавлении товаров, около значка корзины, должно появиться общее количество товаров в корзине.
4. На главной странице – список товаров с изображением, названием, ценой, выбором количества и фильтрацией по категориям. Выбранные товары должны сразу попадать в корзину.
5. При нажатии на товар должна открыться подробная страница товара, где можно добавить товар в корзину.
6. Из корзины можно удалять товары, как поштучно, так и все сразу.
7. Страница оформления заказа. В корзине должна быть кнопка купить, после нажатия должно переходить на новое окно, где будет указана сумма и кнопка купить.
8. Значок поиска должен находиться в шапке. И быть доступен с любого окна.
Обязательно использовать redux и hooks.
Создать репозиторий на github. Коммитить все изменения по мере выполнения (Не пушить все разом в конце).
Срок – 2 дня со дня получения задания.
